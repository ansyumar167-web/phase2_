# Frontend Layouts Skill

## Instructions

1. **Page structure**
   - Use semantic HTML
   - Organize sections logically
   - Ensure responsive layouts

2. **Components**
   - Build reusable UI components
   - Separate server and client logic if using frameworks
   - Implement loading states and error boundaries

3. **Styling**
   - Use Tailwind CSS or CSS modules
   - Apply responsive breakpoints and fluid typography
   - Ensure accessible color contrast

## Best Practices
- Mobile-first design
- Keep components modular and reusable
- Use semantic HTML tags
- Optimize images and assets
- Maintain consistent spacing and typography

## Example Structure
```html
<div class="page">
  <header class="header">
    <h1>Main Title</h1>
  </header>
  <main class="content">
    <section class="section">
      <h2>Section Title</h2>
      <p>Section content</p>
    </section>
    <div class="component">
      <button class="btn">Action</button>
    </div>
  </main>
  <footer class="footer">
    <p>Footer content</p>
  </footer>
</div>
